# Spotify Social Media Analytics – Excel
Built an Excel-based Social Media Intelligence dashboard by consolidating spotify’s social media data to track engagement, reach, and content performance.

## Tools Used
- MS Excel

## Skills Demonstrated
- Data cleaning and transformation
- Pivot Tables and KPI analysis
- Dashboard creation and reporting
- Business insight generation

## Files Overview
- data/: Raw and cleaned datasets
- analysis &  dashboard/: Final Excel dashboard
- insights/: Key findings and recommendations

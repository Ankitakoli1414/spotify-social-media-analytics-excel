Overview
This folder contains the raw and cleaned social media datasets used for analysis and dashboard creation.
The goal is to structure and prepare the data for accurate insights and visualization in Excel.

1. Cleaning & Preparation Steps
   Handling Missing Values
   Removed or filled missing entries in key columns like engagement, reach, and post type.

3. Standardizing Formats
   Converted dates to consistent datetime format.
   Standardized column names and content categories.

4. Data Transformation
   Filtered irrelevant or duplicate records.

5. Validation
   Checked for consistency and accuracy across all relevant fields.
   Ensured numeric fields are ready for aggregation and pivot analysis.
